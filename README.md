# Retro Mac OS 3D Portfolio

A creative portfolio website built with Next.js, featuring an interactive 3D retro Mac OS desktop environment. Built with React Three Fiber, TipTap, and custom Vim editor integration.

## Features

- 🖥️ Interactive 3D retro Mac OS desktop environment
- 📝 Rich text editor with TipTap
- ✅ Todo list application
- 🎮 Falling sand physics simulation
- ⌨️ Vim-style editor integration
- 🎨 Custom cursor and UI elements
- 📱 Responsive design

## Tech Stack

- **Framework**: Next.js 16
- **3D Graphics**: React Three Fiber, Three.js, @react-three/drei
- **Editor**: TipTap, ProseMirror
- **Styling**: CSS Modules
- **Language**: TypeScript

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
npm start
```
